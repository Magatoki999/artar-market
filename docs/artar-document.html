01 — OVERVIEW
システム概要と機能状況

ArtARはアートフリマ向けのAR x NFT x カード決済 x AI接客プラットフォームです。クリエーターが作品を登録するとARページとQRコードが発行され、来場者がQRをかざすとキャラクターがAR空間に現れ、音声で話しかけてきます。2体のキャラクターが自発的に会話するダブルモード、購入するとPolygonブロックチェーンにNFT購入証明が自動発行されます。

04 — API REFERENCE
API一覧（7/12本使用）
GET/api/artist?id={id}
作品情報を取得。email除外。404は期限切れまたは存在しない。
GET/api/artist?list=true
登録済み全作品を新着順で取得（最大100件）。トップページで使用。
POST/api/artist
作品登録。必須: name・bio・artworkName・price。sampleLines・artworkStory・artworkMessage・faqs・dualMode・character2等も保存。90日TTL。
POST/api/chat
シングル/デュアルモード対応。dualMode:true + trigger:'spontaneous'で自発会話生成。dialogue[]形式で返却。セリフサンプル・FAQ反映。
GET/api/upload?filename={name}
R2 Pre-signed URLを発行。.glb→glb/フォルダ、画像→nft/フォルダ。15分有効。
POST/api/purchase
Stripe PaymentIntentを作成してclient_secretを返す。必須: artistId・amount・email。
POST/api/webhook
Stripe Webhook。payment_intent.succeeded→Privy wallet→NFT mint→Resendメール。常に200返却。
GET/api/admin?action=verify&password=xxx
パスワード認証確認。
GET/api/admin?action=stats&password=xxx
売上統計・作品別集計・日別集計・Polygonウォレット残高。
GET/api/admin?action=payments&password=xxx
売上一覧（最大100件・ページネーション対応）。
GET/api/admin?action=artists&password=xxx
登録作品一覧（Redis全件取得）。
POST/api/tts
TTS音声生成。character別にElevenLabs/Gemini/VOICEVOXを切替。preprocessTextで読み間違い防止。

06 — OPERATION MANUAL
クリエーター登録手順
1
登録ページにアクセス
artar-market.vercel.app/register を開く
2
作家情報を入力（必須）
作家名・読み仮名・自己紹介を入力。読み仮名はTTSの自己紹介に使用される。
3
人格深化情報を入力（任意・推奨）
セリフサンプル3つ・制作ストーリー・作品に込めたメッセージ・FAQ3組を入力するとAIの返答が格段にリアルになる。
4
作品情報を入力（必須）
作品名・販売価格（100円以上）・作品画像URL（外部URL）を入力。
5
ファイルをアップロード（任意）
3Dモデル（.glb・50MB以内）とNFT証明画像（jpg/png/webp・10MB以内）をアップロード。未アップロードはキャラデフォルト画像を使用。
6
キャラクター選択・ダブルモード設定
空蝉/夕顔/Dr.Ohma/Acielから選択。ダブルモードONで2体目を選択・GLB2つ目をアップロード可能。2体が自発的に会話する。
7
登録完了 → QRコードを印刷
ARページURLとQRコードが発行される。QRを印刷してブースに設置する。
発行URL: https://artar-market.vercel.app/artist/{id} データ保存期間: 90日（Upstash Redis TTL）

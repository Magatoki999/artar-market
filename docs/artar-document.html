01 — OVERVIEW
システム概要と機能状況

ArtARはアートフリマ向けのAR x NFT x カード決済プラットフォームです。クリエーターが作品を登録するとARページとQRコードが発行され、来場者がQRをかざすとアーティストのキャラクターがAR空間に現れ作品の世界観を体験できます。購入するとStripeでカード決済が完了し、PolygonブロックチェーンにNFT購入証明が自動発行されます。

04 — API REFERENCE
API一覧（5/12本使用）
GET/api/artist?id={id}
作品情報を取得。emailは除外して返す。404は期限切れまたは存在しない。
GET/api/artist?list=true
登録済み全作品を新着順で取得（最大100件）。トップページで使用。重複IDは除外。
POST/api/artist
作品登録。必須: name・bio・artworkName・price(100円以上)。Redisに90日TTLで保存。
POST/api/chat
作家人格AIチャット。artistDataを渡すと作家として回答。visitorLangで9言語自動切替。150字以内。
GET/api/upload?filename={name}
R2 Pre-signed URLを発行。.glb→glb/フォルダ、画像→nft/フォルダ。15分有効。CORSはR2で設定。
POST/api/purchase
Stripe PaymentIntentを作成してclient_secretを返す。必須: artistId・amount・email。
POST/api/webhook
Stripe Webhook。payment_intent.succeeded受信→署名検証→Privy wallet→NFT mint→Resendメール。Stripeには常に200を返す。

06 — OPERATION MANUAL
クリエーター登録手順
1
登録ページにアクセス
artar-market.vercel.app/register を開く
2
作家情報を入力（必須）
作家名・ひとこと自己紹介を入力。自己紹介はAIチャットの人格に使われる。
3
作品情報を入力（必須）
作品名・販売価格（100円以上）を入力。作品画像URLは外部URL（SNS・imgur等）を貼る。
4
ファイルをアップロード（任意）
3Dモデル（.glb・50MB以内）とNFT証明画像（jpg/png/webp・10MB以内）をアップロード。未アップロードの場合はキャラデフォルト画像が使用される。
5
キャラクターを選択
空蝉 / 夕顔 / Dr.Ohma / Aciel から選択。GLBをアップロードした場合はそちらが優先表示される。
6
登録完了 → QRコードを印刷
ARページURLとQRコードが発行される。QRコードを印刷してフリマブースに設置する。
発行URL: https://artar-market.vercel.app/artist/{id} データ保存期間: 90日（Upstash Redis TTL）
